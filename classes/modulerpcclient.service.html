Class Service<serviceDefinition, ResponseContext>

Implements a service from a StreamProducer. The StreamProducer is provided by a streaming protocol, for instance through getGrpcWebClient in the case of the gRPC-Web protocol.

Type parameters

  • serviceDefinition: ServiceDefinition

    The definition of the service, enumerating all the methods and their request and response types.

  • ResponseContext

    The type of the response context that is transmitted along with the response to the client.

nice

  • nice(): NiceService<serviceDefinition>
    • Defined in client/service.ts:160

    Return a "nice" service interface with which it is possible to call RPCs as "normal" JavaScript functions.

    example
    // Before
const service: Service<...> = ...;
service.stream('serverStream', { foo: 'bar' })
  .on('message', ({ response, responseContext }) => { ... })
  .start();
const { response, responseContext } = await service.call('unaryMethod', { foo: 'bar' });

// After
const niceService = service.nice();
niceService.serverStream({ foo: 'bar' })
  .on('message', response => { ... })
  .start();
const response = await niceService.unaryMethod({ foo: 'bar' });
